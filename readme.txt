1. Compiliation
  The game can be compiled by simply typing:
	make build
  
  When recompiling you should instead use the command:
	make

  To remove the executables and build files you can use the following:
 	make clean

2. Controls
  W - Move up
  A - Move left
  S - Move down
  D - Move right

  R - Re-run the level generation
  Q - Quit the game
